Lab 15 - AJAX

Challenge

For this lab, we were tasked with configuring a webpage that featured a button, which would fetch an API once clicked and display information based on the fetched API.

Problems

I kept running into an issue where CORS policy would block AJAX from fetching the JSON from the website, and this was rather frustrating because all of the various websites I visited would run into this issue. Not even a CORS unblocker extension worked, and I didn't want to use it because I couldn't assume that anybody viewing my webpage would have that extension installed. Thus, I eventually resorted to finding a REST API that was CORS-friendly and used that as my API instead.

Reflection

I put a lot of effort into this assignment, since I'm not familiar with APIs at all, and I had a lot of issues fetching JSONs from APIs due to the CORS policy mentioned earlier. At the end of the day, I did get an API to work correctly, but it came after a lot of frustration and effort spent in order to get working.
