Pseudocode for making a meatloaf:

  1. Buy ingredients:
    • eggs
    • ground beef
    • bread crumbs
    • ketchup
    • garlic
    • Worcestershire sauce
  2. Prepare the kitch and baking pans
  3. Preheat the oven
  4. Cut ingredients
  5. Combine ingredients in bowl
  6. Transfer the mixed ingredients into a baking pan
  7. Place into oven and bake
  8. Prepare the glaze
  9. Take the meatloaf out of the oven
  10. Put the glaze on the meatloaf
  11. Return the meatloaf to the oven to bake
  12. Take the meatloaf out of the oven and serve
Pseudocode for Minesweeper:
  1. User chooses the size of the map
  2. Program generates objects on the map
    1. Choose number of mines based on map size
    2. Select random positions of tiles on map
    3. Generate numbers for tiles based on mine positions
  3. Display UI to the player
    1. Obscure tiles
    2. Include counter for mines left on the map
    3. Include timer
  4. Check for mouse click on a tile
    1. If left click: reveal the tile
      1. If the tile is a mine: Game Over
      2. Else: display the tile
      3. If the tile has no number, display all touching tiles with no number
    2. If right click: place a marker on the tile
    3. If all non-mine tiles are revealed: Game Win
